YEONSAM BOT

심플하게 쓰는 서버 안내용 디스코드 봇

yeonsam bot은 서버 입장 안내, 간단한 서버 프로필 수정, 관리자용 안내 문구 편집처럼 자주 쓰는 흐름을 빠르게 다루기 위한 디스코드 봇입니다.

메인 페이지는 소개 중심 구조로 다시 정리했고, 핵심 설명과 다음 행동이 한눈에 읽히도록 여백과 대비를 강화했습니다.

ADD TO DISCORD

디스코드에 추가하는 법

1. 초대 링크 열기

봇 전용 초대 링크를 열고 추가할 서버를 선택합니다.

2. 권한 확인

봇이 필요한 권한을 확인한 뒤 서버에 추가합니다.

3. 슬래시 명령 사용

/yeonsam으로 안내 화면을 확인하고 기능을 시작합니다.

NOTICE

현재는 테스트 단계라 공개 초대 링크는 아직 연결하지 않았습니다.
