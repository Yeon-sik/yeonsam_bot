YEONSAM BOT

서버 안내와 게임 가이드를 더 간편하게 만드는 디스코드 봇

yeonsam bot은 새 멤버 안내, 서버별 프로필 수정, 게임 가이드 탐색까지 디스코드 안에서 자연스럽게 이어지도록 설계된 봇입니다.

복잡한 설정 설명보다 바로 사용할 수 있는 흐름에 집중했습니다. 멤버 온보딩은 더 부드럽게, 게임 정보 탐색은 더 빠르게 정리해줍니다.

FLOW

처음부터 바로 쓰기 쉬운 흐름

1. 멤버 맞이

새로운 멤버가 들어오면 환영 메시지와 함께 필요한 첫 안내를 깔끔하게 전달합니다.

2. 서버 프로필 정리

멤버는 서버에 맞는 닉네임으로 빠르게 프로필을 정리하고, 운영진은 진입 장벽을 낮출 수 있습니다.

3. 가이드 탐색

자주 보는 게임 정보는 디스코드와 웹 문서에서 이어서 확인할 수 있어 필요한 순간에 바로 찾기 쉽습니다.

NOTICE

yeonsam bot은 서버 운영 흐름을 방해하지 않으면서 필요한 안내와 정보를 자연스럽게 연결하는 데 초점을 두고 있습니다.

FEATURES

서버 운영에 바로 도움이 되는 기능

WELCOME 입장 환영 메시지 새 멤버가 들어오는 순간 필요한 안내를 전달해 첫인상을 더 정돈된 경험으로 바꿔줍니다. LIVE
PROFILE 서버별 닉네임 수정 서버 분위기에 맞는 프로필 정리가 쉬워져 멤버 관리와 커뮤니티 적응이 한결 수월해집니다. LIVE
ADMIN 운영진용 안내 관리 서버 성격에 맞게 안내 문구를 손쉽게 조정해 더 일관된 운영 경험을 만들 수 있습니다. LIVE
GUIDE 선택형 게임 가이드 복잡한 검색 없이 필요한 주제를 순서대로 골라 보며 원하는 정보까지 빠르게 도달할 수 있습니다. LIVE
WEB 웹 문서 연동 디스코드에서 보던 가이드를 웹 페이지로 이어서 확인할 수 있어 정리된 정보 탐색에 잘 어울립니다. LIVE
